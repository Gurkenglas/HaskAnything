Hask Anything! is a website aimed at collecting and organizing the collective knowledge of the Haskell community: papers, blog posts, code snippets, gists, StackOverflow answers, Reddit threads, presentations, videos, tutorials and so on.

Add or improve knowledge

Add new content via the website

You can submit new entries for content directly via the website: $categories$

Improve existing content via the website

Every piece of content has a link to its Github source. You can modify the plain-text version of the content directly and propose it as an improvement.

Contribute via Github

The website-based submission process uses Github pull requests behind the scenes. Of course, you can propose pull requests manually, too.
