Add or improve knowledge

Add new content via the website

You can submit new entries for content directly via the website via the Add a new... option in the navigation bar.

Improve existing content via the website

Every piece of content has a link to its Github source. You can modify the plain-text version of the content directly and propose it as an improvement.

Contribute via Github

The website-based submission process uses Github pull requests behind the scenes. Of course, you can propose pull requests manually, too. This is the recommended way of contributing large amounts of content at once.
